## 3. 段落与换行

正常直接输入文字就是段落。

```markdown
这是第一段内容。

这是第二段内容。
```

注意：

- 两段之间要**空一行**
    
- 如果只是想换行，可以直接回车，但一般 Markdown 更常用“空一行分段”
    

---

## 4. 加粗、斜体、删除线、高亮

### 加粗

```markdown
**加粗文字**
```

效果：**加粗文字**

### 斜体

```markdown
*斜体文字*
```

效果：_斜体文字_

### 加粗+斜体

```markdown
***加粗且斜体***
```

效果：_**加粗且斜体**_

### 删除线

```markdown
~~删除线文字~~
```

效果：~~删除线文字~~

### 高亮（Obsidian 常用）

```markdown
==高亮文字==
```

效果：==高亮文字==

---

## 5. 列表

### 无序列表

用 `-` 或 `*` 都可以。

```markdown
- 苹果
- 香蕉
- 橘子
```

效果：

- 苹果
    
- 香蕉
    
- 橘子
    

### 有序列表

```markdown
1. 第一项
2. 第二项
3. 第三项
```

效果：

1. 第一项
    
2. 第二项
    
3. 第三项
    

---

## 6. 任务列表

这个在 Obsidian 里特别实用。

```markdown
- [ ] 学习 Markdown
- [x] 下载 Obsidian
- [ ] 开始记笔记
```

效果：

- [ ] 学习 Markdown  
- [x] 下载 Obsidian  
- [ ] 开始记笔记


说明：

- `[ ]` 表示未完成
    
- `[x]` 表示已完成

## 7. 引用

用 `>` 表示引用内容。

```markdown
> 这是一段引用。
```

效果：

> 这是一段引用。

可以用来记录别人的话、重点句子、书摘等。

## 9. 行内代码

如果只是想在一句话里标出代码，用一个反引号 `` ` `` 包起来。

```markdown
这是 `printf()` 的用法。
```

效果：

这是 `printf()` 的用法。

---

## 10. 代码块

如果要写多行代码，用三个反引号。

````markdown
```cpp
#include <iostream>
using namespace std;

int main() {
    cout << "Hello World" << endl;
    return 0;
}
```
````

效果：

```cpp
#include <iostream>
using namespace std;

int main() {
    cout << "Hello World" << endl;
    return 0;
}
```

你也可以把 `cpp` 换成其他语言，例如：

- `python`
    
- `java`
    
- `javascript`
    
- `html`

## 12. Obsidian 内部链接

这是 Obsidian 很重要的功能。

```markdown
[[欢迎]]
```

[[欢迎]]

作用：链接到另一个笔记。

例如你有一个笔记叫“英语单词”，那么写：

```markdown
[[英语单词]]
```

点击后就能跳转过去。

---

## 13. 链接到某个标题

```markdown
[[Markdown 格式学习笔记#标题]]
```

作用：跳转到某个笔记中的某个标题位置。

## 14. 嵌入笔记或文件

### 嵌入一个笔记

```markdown
![[我的笔记]]
```

### 嵌入图片或文件

```markdown
![[图片.png]]
```

这样可以直接在当前笔记里显示内容，而不只是链接。

---

## 15. 图片

Markdown 普通图片写法：

```markdown
![图片说明](图片地址)
```

例如：

```markdown
![示例图片](https://example.com/test.png)
```

不过在 Obsidian 里，更常见的是直接拖图片进去，Obsidian 会自动帮你生成格式。

## 17. 复选缩进与层级

任务列表和普通列表都可以缩进，表示层级关系。

```markdown
- 学习内容
  - Markdown
  - Obsidian
- 练习内容
  - 写标题
  - 写代码块
```

效果：

- 学习内容
    
    - Markdown
        
    - Obsidian
        
- 练习内容
    
    - 写标题
        
    - 写代码块

## 18. 常用快捷记忆

下面这些是最常用的格式：

| 符号         | 作用            |
| ---------- | ------------- |
| `#`        | 标题            |
| `**文字**`   | 加粗            |
| `*文字*`     | 斜体            |
| `~~文字~~`   | 删除线           |
| `==文字==`   | 高亮            |
| `-`        | 无序列表          |
| `1.`       | 有序列表          |
| `- [ ]`    | 未完成任务         |
| `- [x]`    | 已完成任务         |
| `>`        | 引用            |
| `` `代码` `` | 行内代码          |
| ` ``` `    | 代码块           |
| `[文字](链接)` | 普通链接          |
| `[[笔记名]]`  | Obsidian 内部链接 |
| `![[文件名]]` | 嵌入文件/图片       |
| `---`      | 分割线           |

## 20. 学习建议

刚开始不用一次性全记住。  
先掌握这几个最常用的就够了：

1. 标题
2. 加粗和斜体
3. 列表
4. 任务列表
5. 代码块
6. 内部链接 `[[笔记名]]`

你在实际记笔记时经常用，很快就会熟悉。

## 21. 总结

Markdown 的特点是：

- 简单
- 清晰
- 适合记笔记
- 非常适合 Obsidian

对于 Obsidian 来说，最重要的是：

- 标题
- 列表
- 任务列表
- 代码块
- 内部链接
- 嵌入笔记或图片

先会最基础的，后面边用边学就行。
