## 1.GPIO是什么？
1. 全称为 General Purpose Input Output， 元器件上的一条条小针就是GPIO的[引脚](obsidian://open?vault=my_notes&file=attachments%2F%E5%B5%8C%E5%85%A5%E5%BC%8F%2FSTM32F103C8T6%E5%BC%95%E8%84%9A%E5%AE%9A%E4%B9%89.jpeg)
2. GPIO作为输出时 像是一个开关 高电平灯亮， 低电平灯暗
3. GPIO作为输入时 像是一个眼睛 可以识别外部状态 比如开关是否按下

## 3. GPIO核心工作模式

| 模式名称                       | 解释                                         | 深入了解                                                               | 适用场景                                 |
| -------------------------- | ------------------------------------------ | ------------------------------------------------------------------ | ------------------------------------ |
| 推挽输出<br>(push-pull)        | 大力士模式<br>主动输出高电平 驱动力强                      | 推挽输出并不是只能输出高电平<br>他只是有输出高电平的能力<br>它既可以输出高电平也可以输出低电平 - [[1](点亮LED)] | [[点亮LED]]<br>[[驱动蜂鸣器]]<br>[[模拟通信协议]] |
| 开漏输出      <br>(Open-Drain) | 没劲模式<br>主动拉低至0V 需要外接上拉电阻才能把电压拉上去           | 开漏输出不可以主动输出高电平 但是可以接入外置高电平<br>但是需要查看引脚容忍程度                         | 多设备通信[[I2C]]<br>[[输出5V电平]]           |
| 上拉/下拉输入                    | 默认模式<br>没信号的时候默认拉高(3.3V) 或 拉低(0V) 防止引脚“瞎跳” |                                                                    | 读取按键状态<br>传感器开关信号                    |
| 模拟输入<br>(Analog)           | 直通模式<br>关闭所有数字加工 把引脚直接连到内部ADC(电压测量表)上      |                                                                    | 精确测量电压                               |
| 浮空输入<br>(Floating)         | 悬浮模式<br>引脚似乎浮空 极易受到干扰 必须接好外设               |                                                                    | 专门的通信接收引脚                            |
|                            |                                            |                                                                    |                                      |
